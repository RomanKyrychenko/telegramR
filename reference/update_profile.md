# Update your name and/or bio

Update your name and/or bio

## Usage

``` r
update_profile(client, first_name = NULL, last_name = NULL, bio = NULL)
```

## Arguments

- client:

  TelegramClient instance.

- first_name:

  First name (optional).

- last_name:

  Last name (optional).

- bio:

  About/bio text (optional).

## Value

The API response.
