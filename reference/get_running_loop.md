# get_running_loop function Get or create a running event loop (cluster) for asynchronous operations.

get_running_loop function Get or create a running event loop (cluster)
for asynchronous operations.

The \`get_running_loop\` function simulates getting the current event
loop.

## Usage

``` r
get_running_loop()

get_running_loop()
```

## Value

A cluster object representing the running event loop.

A promise object or NULL.
