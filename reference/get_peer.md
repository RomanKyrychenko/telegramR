# Get Peer Object

Converts the given peer representation into a Peer object (PeerUser,
PeerChat, or PeerChannel). Handles various input types such as integers,
resolved peers, dialogs, etc.

## Usage

``` r
get_peer(peer)
```

## Arguments

- peer:

  The peer object or integer to convert.

## Value

A Peer object (PeerUser, PeerChat, or PeerChannel).
