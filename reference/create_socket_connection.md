# Create a socket connection.

Placeholder function. In practice, use proper R networking packages.

## Usage

``` r
create_socket_connection(
  host,
  port,
  proxy = NULL,
  local_addr = NULL,
  timeout = NULL
)
```

## Arguments

- host:

  Hostname.

- port:

  Port number.

- proxy:

  Parsed proxy settings.

- local_addr:

  Local address.

- timeout:

  Connection timeout.

## Value

A socket object.
