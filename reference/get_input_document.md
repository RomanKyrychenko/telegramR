# Get Input Document

Similar to `get_input_peer`, but for documents.

## Usage

``` r
get_input_document(document)
```

## Arguments

- document:

  The document object to convert.

## Value

An InputDocument object.

## Examples

``` r
if (FALSE) { # \dontrun{
# Assuming document is a Document object
input_doc <- get_input_document(document)
} # }
```
