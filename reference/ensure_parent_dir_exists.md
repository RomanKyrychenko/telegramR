# Ensure the parent directory exists

Ensure the parent directory exists

## Usage

``` r
ensure_parent_dir_exists(file_path)
```

## Arguments

- file_path:

  The file path for which to ensure the parent directory exists.

## Value

None. Creates the parent directory if it does not exist.

## Examples

``` r
if (FALSE) { # \dontrun{
ensure_parent_dir_exists("path/to/some/file.txt")
} # }
```
