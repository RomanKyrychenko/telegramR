# Get Input User

Similar to `get_input_peer`, but for `InputUser`'s alone.

## Usage

``` r
get_input_user(entity)
```

## Arguments

- entity:

  The entity object to convert.

## Value

An InputUser object.

## Examples

``` r
if (FALSE) { # \dontrun{
# Assuming entity is a User object
input_user <- get_input_user(entity)
} # }
```
