# Get Display Name

Gets the display name for the given User, Chat or Channel. Returns an
empty string otherwise.

## Usage

``` r
get_display_name(entity)
```

## Arguments

- entity:

  The entity object (User, Chat, or Channel).

## Value

A character string representing the display name.

## Examples

``` r
if (FALSE) { # \dontrun{
# Assuming entity is a User object
name <- get_display_name(entity)
} # }
```
