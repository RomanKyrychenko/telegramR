# Get Extension

Gets the corresponding extension for any Telegram media.

## Usage

``` r
get_extension(media)
```

## Arguments

- media:

  The media object to analyze.

## Value

A character string representing the file extension.

## Examples

``` r
if (FALSE) { # \dontrun{
# Assuming media is a Document object
ext <- get_extension(media)
} # }
```
