# Convert Bigz Integer/Numeric to Bytes

Convert Bigz Integer/Numeric to Bytes

## Usage

``` r
int_to_bytes(val, length, endian = "big")
```

## Arguments

- val:

  A bigz integer or numeric.

- length:

  The length of the resulting raw vector.

- endian:

  Either "big" or "little".

## Value

A raw vector.
