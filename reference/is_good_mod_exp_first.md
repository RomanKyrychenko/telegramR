# Is Good Mod Exp First

Checks if a modular exponentiation result is good for the first check.

## Usage

``` r
is_good_mod_exp_first(modexp, prime)
```

## Arguments

- modexp:

  A bigz integer.

- prime:

  A bigz integer.

## Value

A logical value.

A logical value.

## Details

Is Good Mod Exp First

Checks if modexp is good for modular exponentiation.
