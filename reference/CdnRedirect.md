# CdnRedirect class

Class for handling CDN redirects.

## Public fields

- `cdn_redirect`:

  The CDN redirect object.

## Methods

### Public methods

- [`CdnRedirect$new()`](#method-CdnRedirect-new)

- [`CdnRedirect$clone()`](#method-CdnRedirect-clone)

------------------------------------------------------------------------

### Method `new()`

Create a new CdnRedirect instance.

#### Usage

    CdnRedirect$new(cdn_redirect = NULL)

#### Arguments

- `cdn_redirect`:

  The CDN redirect object.

------------------------------------------------------------------------

### Method `clone()`

The objects of this class are cloneable with this method.

#### Usage

    CdnRedirect$clone(deep = FALSE)

#### Arguments

- `deep`:

  Whether to make a deep clone.
