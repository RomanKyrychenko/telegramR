# Get Input Channel

Similar to \`get_input_peer\`, but for \`InputChannel\`'s alone.

## Usage

``` r
get_input_channel(entity)
```

## Arguments

- entity:

  The entity object to convert.

## Value

An InputChannel object.

## Examples

``` r
if (FALSE) { # \dontrun{
# Assuming entity is a Channel object
input_channel <- get_input_channel(entity)
} # }
```
