# Check Prime and Generator with Detailed Checks

Checks if the prime and generator are good with detailed validation,
including bit length, primality, and specific modular conditions.

Validates the prime number and generator g for cryptographic purposes.

## Usage

``` r
check_prime_and_good_check(prime, g)

check_prime_and_good_check(prime, g)
```

## Arguments

- prime:

  An integer representing the prime number.

- g:

  An integer representing the generator.

## Value

Raises an error if validation fails, otherwise returns nothing.
