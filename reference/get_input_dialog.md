# Get Input Dialog

Similar to `get_input_peer`, but for dialogs.

## Usage

``` r
get_input_dialog(dialog)
```

## Arguments

- dialog:

  The dialog object to convert.

## Value

An InputDialogPeer object.

## Examples

``` r
if (FALSE) { # \dontrun{
# Assuming dialog is a dialog object
input_dialog <- get_input_dialog(dialog)
} # }
```
